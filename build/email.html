Корпоративная электронная почта

Электронная почта университета имеет адреса в домене @ncfu.ru и доступна для всех сотрудников и подразделений университета.

Личная электронная почта выдается сотруднику университета на базе единой учетной записи по устному требованию сотрудника специалисту ИТ службы.

Электронная почта подразделения создается на основании служебной записки в свободной форме от начальника структурного подразделения с указанием лица, ответственного за прием и получение почты. При именовании таких электронных ящиков отдается предпочтение аббревиатурам расшифровываемых на английском языке.

Электронный почтовый ящик доступен из интернета и локальной сети университета с использованием веб доступа по адресу https://mail.ncfu.ru. Для работы с почтовым ящиком на компьютере можно использовать более функциональное программное обеспечение Microsoft Outlook. Электронный почтовый ящик также доступен на мобильных устройствах на базе Android и IOS.

Наличие электронной почты обязательно для доступа к некоторым сервисам, например, к серверу веб конференций Cisco WebEx.

Сервис корпоративной электронной почты реализован на платформе Microsoft Exchange 2013.

Инструкции по настройке:

  • Инструкция по настройке почтовых клиентов для корпоративной почты в Android
  • Инструкция по настройке почтового клиента для корпоративной почты в iOS
  • Инструкция по настройке корпоративного почтового клиента Microsoft Outlook 2013
  • Инструкция по созданию контактов в WEB-версии клиента Outlook
  • Инструкция по созданию собраний в почтовом клиенте Outlook 2013
  • Инструкция по работе с общей электронной почтой
